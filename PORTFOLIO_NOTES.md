# Portfolio Notes

This repository is a sanitized portfolio demo derived from previous e-commerce admin system experience.

Before publishing changes, keep the repository free of:

- Real customer names, phone numbers, addresses, order numbers, or payment data
- Production API domains, internal IPs, and company-specific domains
- API keys, tokens, app IDs, app secrets, cloud storage keys, and database credentials
- Original client branding, logos, or confidential business rules

The demo uses mock data by default through `VITE_USE_MOCK_API=true`.
